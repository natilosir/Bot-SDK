# Bot-SDK
A professional SDK BOT Telegram for PHP

## Requirements

- PHP >= 5.4
- Composer

## Installation

You can install this SDK package via Composer:

```bash
composer require natilosir/Telegram-Bot-SDK
php vendor/natilosir/bot/install.php
```
